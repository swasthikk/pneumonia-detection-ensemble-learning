# 🫁 Pneumonia Detection using Deep Learning

## 🚀 Overview

This project focuses on detecting pneumonia from chest X-ray images using advanced deep learning techniques. We implemented transfer learning models and ensemble methods to achieve high classification performance.

---

## 🧠 Key Features

* Transfer Learning (EfficientNetB0, DenseNet121)
* Stratified K-Fold Cross Validation
* Class Imbalance Handling
* Ensemble Learning
* AUC-based Evaluation
* Confusion Matrix & Performance Metrics

---

## 📊 Results

* DenseNet AUC: ~0.97
* Ensemble AUC: ~0.97
* High recall ensuring minimal false negatives

---

## 🏗️ Methodology

1. Data preprocessing using TensorFlow pipelines
2. Transfer learning with pretrained CNNs
3. Fine-tuning top layers
4. Ensemble prediction using weighted averaging

---

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* ROC-AUC

---

## ⚠️ Challenges

* Class imbalance in dataset
* EfficientNet underperformance
* High computational cost
